# Webpack Task 0

This project demonstrates a basic Webpack setup with jQuery integration.

## Installation & Usage

1. Install dependencies:
   ```sh
   npm install
   ```
2. Build the project:
   ```sh
   npx webpack --entry ./src/index.js --output-path ./dist --mode development
   ```
3. Open `dist/index.html` in your browser to view the result.

## Features
- Uses Webpack and webpack-cli as devDependencies
- Uses jQuery as a dependency
- Dynamically adds three paragraphs to the page using jQuery
- Output files are generated in the `dist` folder
