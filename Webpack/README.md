# Webpack Project

This project contains several tasks demonstrating the use of Webpack with JavaScript and TypeScript, following Holberton School requirements.

## Requirements
- All files are compatible with Ubuntu 20.04 LTS, Node 20.x.x, and npm 9.x.x
- Allowed editors: vi, vim, emacs, Visual Studio Code
- All files end with a new line
- Each task contains its own README.md and required files

## Structure
- `task_0/` – Basic Webpack setup with jQuery
- `task_1/` – ...
- `task_2/` – ...
- etc.

## Usage
Navigate to the desired task folder and follow the instructions in its README.md to install dependencies and run the project.
