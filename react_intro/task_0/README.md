# Dashboard Task 0

This folder contains the Vite-based React dashboard app for Holberton School.

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Run tests:
   ```sh
   npm test
   ```

## Assets
- `src/assets/holberton-logo.jpg` - Holberton logo
- `src/assets/close-button.png` - Close button image
- `public/favicon.ico` - Favicon
