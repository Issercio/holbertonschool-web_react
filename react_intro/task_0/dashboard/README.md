# Holberton School Dashboard

This is a basic React application built with Vite for the Holberton School dashboard project.

## Features
- Custom favicon and logo
- Styled header, body, and footer
- Jest setup for testing

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Run tests:
   ```sh
   npm test
   ```

## Project Structure
- `src/` - React source files
- `public/` - Static assets (favicon)
- `src/assets/` - Images (logo, close button)
